# 📌 Student Task Management Application

A simple and efficient **Task Management Web App** built using **HTML, CSS, and JavaScript**.  
This project helps students organize their daily tasks, track progress, and stay productive.

---

## ✨ Features
- ➕ Add new tasks  
- ✅ Mark tasks as completed  
- 🗑️ Delete tasks easily  
- 💾 Save tasks in **Local Storage** (tasks remain after page refresh)  
- 📱 Responsive design for both desktop and mobile  

---

## 🎯 Future Enhancements
- Task categories (Work, Study, Personal)  
- Due dates & reminders  
- Priority levels with color coding  
- Dark mode toggle  
- Progress tracker
